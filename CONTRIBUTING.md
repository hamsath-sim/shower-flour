# Contributing to Shower Flour

We're excited that you're interested in contributing! Here are some guidelines to help you get started.

## How to Contribute

1. **Fork the Repository:** Create your own copy of the project.
2. **Create a Branch:** `git checkout -b feature/your-feature-name`
3. **Make Changes:** Implement your feature or fix.
4. **Commit Changes:** `git commit -m "Add some feature"`
5. **Push to Branch:** `git push origin feature/your-feature-name`
6. **Open a Pull Request:** Submit your changes for review.

## Code Standards

- Follow the existing project structure and styling.
- Ensure your code is clean, well-commented, and passes linting.
- Respect the brand colors and design patterns.

## Team Workflow

- Use GitHub issues to track bugs and features.
- Assign yourselves to tasks to avoid duplicate work.
- Reach out to the lead developer if you have questions about the architecture.

Thank you for helping us grow Shower Flour!
